# AZ-104 Certification Notes
## Chapter 9.6 - MobileApp

### Monitor VMs via Azure Mobile App
You can install the **Azure Mobile App**, and you can monitor your VMs on the go. You can access basic functionality to control your VMs, services, and even access the Azure Cloud Shell.

### Practice Quiz
* What is the Azure Mobile App?
	* **Azure Mobile App allows you to store app data in the cloud or on-premises, and monitor your VMs on the go, etc**
