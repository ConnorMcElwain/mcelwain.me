# MS-102 Certification Notes
## Day 8.6 - Self Service Password Reset

### Self Service Password Reset
* Entra ID - Configure Settings
* Password Writeback - Configure On-premises integration
* Azure AD Connect - Configure Password Writeback
* User Experience - Can perform their own password reset

### Password Writeback
* Check Entra ID On-Premises integration settings
* Configure Azure AD on premises
* Configure Entra ID On Premises integration settings
* Microsoft 365 Premium, Entra ID P1 or P2 License required for user account that writes back

### Configure Azure AD On-Premises
* Stop Synchronization Service
* Open Azure AD Connect
* Go through Configuration wizard and change settings
* Verify Configuration in Entra ID
