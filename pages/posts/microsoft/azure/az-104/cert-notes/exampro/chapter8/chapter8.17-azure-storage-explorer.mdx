# AZ-104 Certification Notes
## Chapter 8.17 - Azure Storage Explorer

### Azure Storage Explorer
A **standalone app** that makes it easy to work with Azure Storage data on Windows, macOS, and Linux. You can create Blob containers, upload files, create snapshots of Disk, and more!

* What is a standalone app that makes it easy to work with Azure Storage data on Windows, macOS, and Linux?
	* Azure Storage Explorer
