# AZ-104 Certification Notes
## Chapter 7.4 - ARM Template Parameters

### ARM Template - Parameters
* Parameters
	* Allows you to pass variables to your ARM template
* type
	* The expected data type of the input value
		* **string**, **securestring**, **int**, **bool**, **secureObject**, and **array**
* defaultValue
	* If a value isn't provided, it will be set to this value (defaultValue)
* allowedValues
	* An array of allowed values
* minValue
	* The minimal possible value
* maxValue
	* The maximum possible value
* minLength
	* The maximum length of characters or array
* maxLength
	* The maximum length of characters or array
* description
	* The description that will be displayed in the Azure Portal

### Practice Quiz
* Which ARM Template parameter sets the maximum possible value?
	* **maxValue**
	* defaultValue
	* allowedValues
	* maxLength

* What is the defaultValue in ARM Template parameters?
	* defaultValue is used if no value is provided it will be set to this value.
