## Chapter 2.7 - Azure AD Connect

### Azure AD Connect
Azure AD Connect is a hybrid service to connect your on-premise Active Directory to your Azure Account. Azure AD Connect allows for seamless Single Sign On from your on-premise workstation to Microsoft Azure.

* Azure AD Connect has the following features:
	* Password hash synchronization
		* Sign-in method, synchronizes a hash of a users on-premises AD password with Azure AD
	* Pass-through authentication
		* Sign-in method, allows users to use the same password on-premises and in the cloud
	* Federation integration
		* Hybrid environment using an on-premises AD FS infrastructure, for certificate renewal
	* Synchronization
		* Responsible for creating users, groups, and other objects, ensures on-prem and cloud data matches
	* Health Monitoring
		* Robust monitoring and provide a central location in the Azure portal to view this activity
