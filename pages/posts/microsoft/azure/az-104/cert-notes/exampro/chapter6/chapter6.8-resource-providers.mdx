# AZ-104 Certification Notes
## Chapter 6.8 - Resource Providers

### Resource Providers
In order to **use** Azure resources, you have to **register Resource Providers**. Many resource providers are registered by default for you with your Subscription.
* You can register Resource Providers under your subscription

* How do you use Azure resources?
	* You have to register for the Resource Providers. Many Resource Providers are registered by default for you with your Subscription.
