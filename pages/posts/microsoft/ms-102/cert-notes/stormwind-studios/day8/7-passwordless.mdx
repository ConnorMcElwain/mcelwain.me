# MS-102 Certification Notes
## Day 8.7 - Passwordless

### Passwordless (or password + not an official term)
* FIDO2 security key
	* USB security key
* Microsoft Authentication
	* Microsoft's official authenticator mobile app
* SMS
	* Code/temp password that's texted to you
* Temporary Access Pass
	* One time password
* Hardware OATH tokens (Preview)
* Third-party software OATH tokens
* Voice call
	* Receive a voice call that verbally provides you a code
* Email OTP (one time password)
* Certificate-based authentication

### Authentication Strengths
* Combination of authentication methods that can be used
	* Multifactor Authentication
	* Passwordless MFA
	* Phishing-resistant MFA
