# MS-102 Certification Notes
## Day 7.10 - Troubleshooting Tools

### Troubleshooting Tools
* Remote Connectivity Analyzer
	* Several connectivity tests validating configuration and documenting errors
	* Run from https://testconnectivity.microsoft.com
	* Tests run in the browser
* Message Analyzer
	* Tool to paste in header information from undeliverable email or error messages with headers
	* Run from the browser
* Support Recovery and Assistant
	* SARA - installed and run from the client having the issue
	* Runs tests and generates detailed messages and logs
