# Microsoft Endpoint Manager Notes
## Chapter 5.5 - Autopilot FAQ's and Troubleshooting Tips

### Windows Autopilot - FAQ's
* Key Troubleshooting Activities
	* Review configuration: Has Azure Active Directory and Microsoft Intune (or an equivalent MDM service) been configured as specified in the configuration requirements?
	* Check network connectivity: Can the device access the service described in the networking requirements?
	* Autopilot out-of-box experience (OOBE) behavior: Are the expected OOBE screens displayed?
	* Is the Azure AD credentials page customized with organization-specific details as expected?
	* Azure AD join issues: Is the device able to join Azure Active Directory?
	* MDM enrollment issues: Is the device able to enroll in Microsoft Intune (or an equivalent MDM service)?
