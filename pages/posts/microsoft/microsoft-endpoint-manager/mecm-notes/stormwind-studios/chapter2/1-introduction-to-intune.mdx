# Microsoft Endpoint Manager Notes
## Chapter 2.1 - Introduction to Intune

### What is Intune?
* Cloud based Device Management
* Configuration for personal and company-owned devices
* Application Management
* Access - Security - Compliance

### Why Intune?
* Compatible with all devices
* Ease of access in the cloud
* Security management
* Software and update management

### Terminology (new vs. Old)
* Co-management used to refer to any mixture of on-prem and cloud
* Co-management is now strictly for Microsoft on-prem and cloud
* Co-exist is for Microsoft on-prem with a 3rd party cloud solution
